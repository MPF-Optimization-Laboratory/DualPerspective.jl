Preconditioning

Each iteration of the preconditioned Newton CG method approximately solves the trust-region subproblem \[ \min_{s} \{\tfrac{1}{2} \langle s, H_k s \rangle + \langle \nabla f_k,s \rangle \mid \|s\|_M \leq \Delta\} \] where \(\nabla f_k\) and \(H_k\) are the current gradient and Hessian, and \(M\approx H\) is a positive definite preconditioner. For the KL regularized LS problem, \[ H_k = A S_k A^T + \lambda I, \quad S_k:= X_k - x_k x_k^T \] where \(x_k\) is the current primal iterate and \(X_k=\operatorname{Diag}(x_k)\).

We consider these preconditioners:

  1. Diagonal, constant: \(M = \operatorname{Diag}(AA^T) + \lambda I\)
  2. Diagonal, variable: \(M = \operatorname{Diag}(AS_kA^T) + \lambda I\)
  3. Cholesky, constant: \(M = AA^T + \lambda I\)

The first two preconditioners have complexity \(O(mn)\) to compute and \(O(m)\) to apply. The third has cubic complexity, but that cost can be amortized over the Newton iterations.
